# Wanderly

A travel discovery app built with React. Browse destinations, explore traveler profiles, and book places — all in one place.

🔗 **Live Demo:** [wanderly.vercel.app](https://wanderly.vercel.app)

---

## Features

- Browse and filter travel destinations by type, price, and rating
- View detailed place pages with booking form
- Explore traveler profiles fetched from a real API
- Authentication with protected routes
- Responsive layout

## Tech Stack

- React 18
- React Router v6
- React Hook Form + Yup
- Vite
- CSS (no UI libraries)

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── api/          # API calls
├── auth/         # Auth context and protected routes
├── components/   # Reusable components
├── data/         # Static places data
├── layouts/      # Root layout
├── pages/        # Page components
└── routes/       # Router config
```
